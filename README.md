# BridgeLabz Training Backend

A comprehensive repository containing backend development projects completed during the BridgeLabz Training Program.

This repository showcases hands-on learning of **Java Backend Development** using **Servlets, JSP, Spring Framework, Spring Boot, REST APIs, JDBC, PostgreSQL, Maven, and related backend technologies**. Each project is maintained in a separate Git branch for better organization and easy navigation.

---

# 📌 Overview

The primary objective of this repository is to strengthen backend development skills by building real-world Java applications and understanding modern backend frameworks and database integration.

### Learning Objectives

- Build Java backend applications
- Understand Spring Framework and Spring Boot
- Develop RESTful APIs
- Learn JDBC and PostgreSQL integration
- Implement authentication and authorization
- Apply MVC architecture
- Manage projects using Maven
- Practice Git and GitHub workflows

# ▶️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/BridgeLabz-Training-Backend.git
```

---

## Switch to a Project Branch

Example:

```bash
git checkout Greeting_Servlet_Application1
```

or

```bash
git checkout Employee_Payroll_App
```

---

## Build the Project

For Maven projects:

```bash
mvn clean install
```

Run the application:

```bash
mvn spring-boot:run
```

For Servlet projects, deploy the generated WAR file on **Apache Tomcat**.
