# Greeting Servlet Application (Without Login)

A simple Java web application that generates personalized greeting messages using **Java Servlets**, **JSP**, and the **Spring Framework**. This project demonstrates the **MVC (Model-View-Controller)** architecture without implementing user authentication.

---

# 📌 Project Overview

This application allows users to enter their name and receive a personalized greeting message. It demonstrates the integration of Servlets, JSP, and the Spring Framework while following the MVC design pattern.

---

# ✨ Features

- Personalized Greeting Message
- Spring Dependency Injection (IoC)
- MVC Architecture
- Servlet-Based Request Handling
- JSP Frontend
- Clean and User-Friendly Interface

# ▶️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/Muskankapoor07/BridgeLabz-Training-Backend.git
```

## Switch to the Project Branch

```bash
git checkout Greeting_Servlet_Application1
```

## Run the Project

1. Import the project into **IntelliJ IDEA**.
2. Configure **Apache Tomcat**.
3. Deploy the project.
4. Start the Tomcat server.
5. Open the application in your web browser.
